CodeAtlas is a full-stack SaaS application that converts GitHub repositories into AI-powered knowledge maps. By combining Google Gemini's language understanding with semantic vector search and interactive 3D visualization, CodeAtlas helps developers and teams understand unfamiliar codebases instantly — without reading every file manually.

## What CodeAtlas does

When you submit a GitHub repository URL, CodeAtlas runs it through a five-stage AI pipeline:

1. **Clones** the repository securely
2. **Scans and indexes** all meaningful source files
3. **Analyzes** the architecture using Google Gemini to identify domain pillars
4. **Audits** for security vulnerabilities against OWASP Top 10 (runs within the analysis phase)
5. **Persists** structured documentation nodes with vector embeddings for semantic search

The result is a navigable knowledge map — organized into architectural pillars, security findings, and a searchable documentation tree — available immediately through the web UI or REST API.

## Tech stack

CodeAtlas is built with a modern, type-safe stack:

| Layer | Technology |
|---|---|
| Frontend | Next.js 16, React 19, Tailwind CSS, shadcn/ui, Three.js |
| Backend | Node.js, Express.js, TypeScript |
| AI | Google Gemini API, LangChain |
| Database | Supabase (PostgreSQL) with vector embeddings |
| Auth | Supabase Auth with JWT validation |
| Validation | Zod schemas shared across frontend and backend |
